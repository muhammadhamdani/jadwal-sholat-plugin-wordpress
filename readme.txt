Plugin WordPress untuk menampilkan jadwal sholat dari api.myquran.com dengan tampilan responsif dan shortcode.

== Description ==

**Jadwal Sholat ** adalah plugin yang menampilkan jadwal sholat harian dari API MyQuran.com.  
Plugin ini cocok untuk website masjid, sekolah Islam, lembaga dakwah, atau situs pribadi muslim.  

✨ **Fitur Utama**  
- Menampilkan jadwal sholat untuk hari ini  
- Pilihan kota dari seluruh Indonesia (data dari api.myquran.com)  
- 4 tema desain berbeda: Modern, Islamic, Minimal, Dark Mode  
- Countdown otomatis menuju sholat berikutnya  
- Auto-refresh setiap 1 menit  
- Tampilan responsive (mobile-friendly)  
- Admin panel untuk pengaturan default  
- Shortcode `[jadwal_sholat]` mudah digunakan  
- Caching data untuk performa optimal  

== Installation ==

1. Download plugin ini.  
2. Upload ke folder `/wp-content/plugins/` atau install melalui menu **Plugins → Add New**.  
3. Aktifkan plugin melalui menu **Plugins** di WordPress.  
4. Masuk ke **Pengaturan → Jadwal Sholat** untuk memilih kota default.  
5. Tambahkan shortcode `[jadwal_sholat]` pada halaman atau widget.  

== Frequently Asked Questions ==

= Bagaimana cara menampilkan jadwal di halaman? =  
Gunakan shortcode:  

[jadwal_sholat]

= Apakah bisa memilih kota sendiri? =  
Ya, tersedia pilihan semua kota di Indonesia sesuai database api.myquran.com.

= Apakah plugin ini gratis? =  
Ya, plugin ini open-source dengan lisensi GPLv2 atau yang lebih baru.
